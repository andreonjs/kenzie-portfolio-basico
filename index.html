GitHub

Para iniciarmos, o Github é identificado como uma rede social de código, ou seja, é uma uma plataforma que intermedia a participação das pessoas para a arquitetura de projetos a fim de serem usados por toda a demanda gratuitamente.

Utilizo diariamente a ferramenta a fim de registrar alguns projetos relevantes. Geralmente cada projeto é contemplado com um volume básico de commits, com o objetivo de manter apenas uma convivência rotineira, ou seja, uma experiência elementar.

React JS

Sem dúvida o React JS é a biblioteca/framework mais famosa da atualidade. Ficando bem na frente do Vue.js e Angular, a comunidade auxilia com uma quantidade importante de publicações importantes, acelerando a densidade na curva de aprendizado.

Realizei a utilização de algumas plataformas eficazes que auxiliam na implementação de interfaces, e além disso, na complementação elementar dos quesitos interatividade e responsividade, para a construção de aplicações modernas.

Exemplos de Plataformas: CDNJS, Bootswatch, Reactstrap, Flux, Material-UI, Formik, Babel JS e Materialize CSS.

Ruby vs Python

O Ruby é uma linguagem de programação iniciada no Japão em meados dos anos 90. Ela foi codificada por Yukihiro Matsumoto. É uma linguagem interpretada de multiparadigma, e com tipagem forte e dinâmica, e além disso, possui gerenciamento de memória automático.

Realizei projetos iniciantes em Ruby/RoR, e senti motivação em me aprofundar a fim de me tornar especialista na tecnologia. Apesar disso, mesmo tenso sido seduzido pelo hype rubista, percebi um desnível na demanda de mercado de acordo com pesquisas recentes, não sendo tão viável quanto antigamente. Com isso, tive a oportunidade de observar a linguagem Python, que possui tipagem forte e dinâmica, e possui alta demanda de mercado.

Plataformas Rubistas: Ruby Core, Dev Docs, API Dock, Ruby Gems, TTY e RoR.
